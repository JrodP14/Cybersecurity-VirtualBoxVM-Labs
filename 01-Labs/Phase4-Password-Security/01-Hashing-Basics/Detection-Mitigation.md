# Detection & Mitigation

## Overview

This document covers defensive password-security practices, monitoring concepts, and mitigation strategies related to password attacks and weak authentication security.

---

# Common Threats

Potential threats include:
- brute-force attacks
- password spraying
- credential stuffing
- weak password usage

---

# Defensive Monitoring

Security teams should monitor:
- failed login attempts
- account lockouts
- suspicious authentication activity
- abnormal login locations

---

# Common Mitigations

| Mitigation | Purpose |
|---|---|
| MFA | Reduce account compromise risk |
| Password Policies | Improve password quality |
| Account Lockouts | Prevent brute-force attacks |
| Monitoring | Detect suspicious activity |

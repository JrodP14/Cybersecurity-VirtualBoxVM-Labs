# Cybersecurity-VirtualBoxVM-Labs(In Progress)

## Overview

This repository documents the hands-on cybersecurity labs, projects, scripts, reports, and technical documentation completed throughout my cybersecurity training at NexGenT along with my own research.

The labs were built using VirtualBox-based environments and focus on developing practical offensive security, defensive security, digital forensics, scripting, and system administration skills.

The repository is designed to demonstrate:
- practical cybersecurity workflows
- technical documentation practices
- offensive and defensive security concepts
- digital forensic recovery procedures
- scripting and automation fundamentals
- vulnerability assessment methodologies
- network and traffic analysis techniques

The environment primarily utilizes:
- VirtualBox
- Kali Linux
- Metasploitable2
- Windows virtual machines
- OpenVAS
- Wireshark
- Sysinternals
- Nmap

# Password Policy Analysis

## Overview

This document covers password-policy concepts and defensive password-security considerations reviewed during the lab.

---

# Strong Password Characteristics

Strong passwords should include:
- sufficient length
- complexity
- uniqueness
- randomness

---

# Common Weaknesses

Weak passwords may involve:
- reuse
- predictable patterns
- short length
- dictionary words

---

# Common Security Recommendations

| Recommendation | Purpose |
|---|---|
| MFA | Additional authentication layer |
| Password Managers | Secure storage |
| Long Passphrases | Improved complexity |
| Account Lockouts | Reduce brute-force attacks |

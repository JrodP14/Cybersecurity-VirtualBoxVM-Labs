# Hashcat Password Cracking

## Overview

This lab focused on password-auditing workflows, hash-cracking concepts, attack modes, and wordlist analysis using Hashcat within the isolated cybersecurity home lab environment.

The purpose of this lab was to develop foundational password-auditing and authentication-security skills commonly used in:
- penetration testing
- password auditing
- red team operations
- security assessments
- defensive password analysis

# Hashing Basics

## Overview

This lab focused on password hashing concepts, Linux password storage, hash identification, salting techniques, and defensive password-security practices within the isolated cybersecurity home lab environment.

The purpose of this lab was to develop foundational authentication and password-security skills commonly used in:
- SOC operations
- IAM/security administration
- password auditing
- incident response
- Linux administration
- defensive security analysis

---

# Lab Objectives

The primary objectives of this lab were:

- Understand password hashing concepts
- Analyze Linux password hashes
- Identify common hash formats
- Review salting techniques
- Practice OpenSSL hashing
- Understand password-security risks
- Review defensive password policies

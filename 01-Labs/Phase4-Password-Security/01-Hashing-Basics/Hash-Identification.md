# Hash Identification

## Overview

This document covers identifying common password-hash formats and recognizing hashing algorithms during password-security analysis.

---

# Common Identification Tools

| Tool | Purpose |
|---|---|
| hashid | Identify hash types |
| hash-identifier | Detect algorithms |

---

# Example Usage

```bash
hashid hashes.txt
```

---

# Common Hash Formats

| Type | Example Characteristics |
|---|---|
| MD5 | 32 characters |
| SHA1 | 40 characters |
| SHA256 | 64 characters |
| NTLM | 32 hex characters |

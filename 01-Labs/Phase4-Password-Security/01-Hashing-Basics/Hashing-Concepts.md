# What Is Hashing?

Hashing converts data into a fixed-length value using a mathematical algorithm.

Hashing is designed to be:
- deterministic
- one-way
- fast
- difficult to reverse

# Password Hashing

Passwords should be:
- hashed
- salted
- securely stored

rather than stored in plaintext.

# One-Way Functions

Hash functions are designed to:
- generate unique outputs
- prevent reverse recovery
- validate integrity
